# 🏗 Architecture

```text
                    👨‍🎓 Student
                         │
                         ▼
         ┌────────────────────────────┐
         │     HTML • CSS • JS UI     │
         │   Responsive Registration  │
         └────────────────────────────┘
                         │
                Form Validation
                         │
                         ▼
                  Fetch API (JSON)
                         │
                         ▼
         ┌────────────────────────────┐
         │     Node.js + Express      │
         │        REST API            │
         └────────────────────────────┘
                         │
                  SQL Queries
                         │
                         ▼
         ┌────────────────────────────┐
         │       PostgreSQL DB        │
         │ Student Records & Storage  │
         └────────────────────────────┘
```

```text
Students
│
├── 🆔 id
├── 👤 first_name
├── 👤 last_name
├── 🎂 dob
├── 📧 personal_email
├── 🎓 university_email
├── 📚 course
├── 🏫 current_year
├── 🎓 graduation_year
├── 🏠 residence_type
├── 🛏 hostel_block
├── 📖 subjects
```
